# אלגוריתמים 1 - סמסטר ב' התשפ"ד (ספטמבר 2024), מועד ב'

**שאלה 1: (20 נקודות)**
נתונה מחרוזת $S=a_{1}a_{2}...a_{n}$ מאורך $n$, כאשר התווים של $S$ הם מספרים שלמים. המטרה היא לחשב את אורך תת הסדרה היורדת "הרציפה" הארוכה ביותר ב-$S$.
תת סדרה כזו מוגדרת כך: זוהי סדרה $a_{k_1}, a_{k_2}, ...$ כך שלכל $j=i+1, i\ge1$, מתקיים $k_{j}-k_{i}=1$ וגם $a_{k_i} > a_{k_j}$.
*   **סעיף א' (10 נקודות):** תארו נוסחה רקורסיבית המבטאת את אורך תת הסדרה היורדת "הרציפה" הארוכה ביותר ב-$S$. הסבירו את נכונות הנוסחה שלכם.
*   **סעיף ב' (10 נקודות):** תארו אלגוריתם תכנות דינאמי יעיל ככל האפשר, המבוסס על הנוסחה שתיארתם בסעיף א', שמחזיר את אורך תת הסדרה הנ"ל ב-$S$. הסבירו את נכונות האלגוריתם (אין צורך להוכיח פורמאלית), ונתחו זמן ריצה.
*(דוגמא: עבור $S = 4, 19, 10$ האלגוריתם יחזיר את הערך 3 המתאים לתת הסדרה).*

**שאלה 2: (20 נקודות)**
יהי $G=(V,E)$ גרף לא מכוון וקשיר, ותהי $w:V\rightarrow\mathbb{R}$ פונקציית משקל על הצמתים. משקל מדורג של עץ $T$ מוגדר להיות:
$W_d(T) = \sum_{v \in V} w(v) \cdot deg_T(v)$
כאשר $deg_T(v)$ מציין את הדרגה (מספר שכנים) של קודקוד $v$ בעץ $T$.
הציעו אלגוריתם יעיל אשר מחזיר עץ פורש של הגרף $G$ בעל משקל מדורג מינימאלי. הוכיחו את נכונות האלגוריתם שלכם ונתחו את זמן הריצה שלו.

**שאלה 3: (20 נקודות)**
$G=(V,E)$ גרף מכוון וקשיר, ותהי $w:E\rightarrow\mathbb{R}$ פונקציית משקל על הקשתות. כמו כן נתון צומת $v_{0}\in V$. הציעו אלגוריתם יעיל המוצא עבור כל זוג צמתים $u,v\in V$ את משקל המסלול הקל ביותר מ-$u$ ל-$v$ העובר דרך $v_{0}$. על האלגוריתם לרוץ בזמן $O(|V||E|)$. הוכיחו את נכונות האלגוריתם שלכם.

**שאלה 4: (20 נקודות)**
נתונה רשת זרימה $G=(V,E)$ עם מקור $s$ ובור $t$, כאשר הקיבולות על הקשתות הם מספרים שלמים, וכמו כן נתונה קבוצת קשתות $F\subseteq E$.
תארו אלגוריתם יעיל הקובע האם לכל חתך $s-t$ מינימום, $(S,T)$, מתקיים כי כל הקשתות ב-$F$ חוצות את $(S,T)$.
הסבירו את נכונות האלגוריתם (אין צורך לתת הוכחה פורמאלית) ונתחו את זמן הריצה שלו.

**שאלה 5: (20 נקודות)**
ב-Reservoir sampling רוצים להגריל מדגם אקראי של $k$ איברים מתוך stream של $n$ איברים $(k<n)$. מגיעים $n$ איברים ברצף, ואין אפשרות להחזיק את כולם ברשימה משום ש-$n$ גדול מדיי, אך רוצים לחלץ מתוכם מדגם אקראי.
להלן אלגוריתם הבונה מדגם כזה לתוך מערך $R$:
1. Initialize an Array $R[1,...,k]$ containing the first $k$ elements of the input $x_1, ..., x_k$. This is the reservoir.
2. For each new input $x_i$, generate a random index $j$ uniformly in $[1,...,i]$.
   If $j\in\{1,...,k\}$, then set $R[j]=x_i$.
   Otherwise, discard $x_i$.
3. Return $R$ after all the input $x_1, ..., x_n$ is processed.

המטרה היא להראות שכל איבר מתוך $x_1,...,x_n$ נבחר לתוך $R$ בהסתברות $k/n$ בדיוק. בשאלה זו נראה את זה רק עבור איברים מאינדקס $k+1$ ומעלה.
*   **סעיף א' (8 נקודות):** הראו שהאיבר האחרון $x_n$ נמצא ב-$R$ בהסתברות $k/n$.
*   **סעיף ב' (12 נקודות):** הראו שכל איבר $x_i$ נמצא ב-$R$ בהסתברות $k/n$, עבור כל אינדקס $i \ge k+1$.
